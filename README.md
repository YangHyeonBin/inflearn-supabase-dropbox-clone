# Dropbox Clone 프로젝트

## 기술 스택

-   Next.js
-   Supabase
-   TailwindCSS
-   React Query

## 기능

-   드래그 앤 드랍으로 파일 업로드
-   클릭하여 파일 업로드
-   여러 개의 파일 한번에 업로드
-   파일 리스트: 이미지 미리보기, 파일명, 타입, 크기, 최근 수정일 표시
-   파일 검색
-   파일 삭제
-   파일 다운로드
